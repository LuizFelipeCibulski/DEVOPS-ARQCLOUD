# 🚀 Pós Tech - DevOps e Arquitetura Cloud

Repositório com minhas anotações, exercícios e projetos da Pós-Graduação em **DevOps e Arquitetura Cloud** (FIAP + Alura).

> 10 meses, 360 horas, para virar um DevOps de verdade (ou pelo menos parar de dar `chmod 777` em tudo) :)

📎 [Site oficial do curso](https://postech.fiap.com.br/curso/devops-e-arquitetura-cloud)

---

## 📚 O que rola em cada fase

### Fase 1 — Cultura DevOps e Arquitetura Cloud
Cultura DevOps, CI/CD, arquitetura cloud (AWS/Azure/GCP), IaaS/PaaS/SaaS/FaaS, do monolito ao microsserviço, AWS (VPC, EC2, ECS, EKS, ELB, RDS, IAM...).

### Fase 2 — Escalabilidade e Conteinerização
Docker (imagens, volumes, compose), Kubernetes do básico ao avançado (Helm, Karpenter, KEDA, Blue/Green, Canary, RBAC), escalabilidade horizontal/vertical, Nginx/Apache/IIS e balanceamento de carga.

### Fase 3 — Pipelines e Segurança na Cloud
CI/CD com GitHub Actions, Infraestrutura como Código (Terraform, OpenTofu, Terragrunt), DevSecOps (SAST/DAST, SIEM, LGPD), segurança na cloud (IAM, MFA, Zero Trust, CSPM/CWPP/CASB).

### Fase 4 — Observabilidade e Monitoramento
Os 3 pilares (logs, métricas, traces), Prometheus/Grafana/Loki/Jaeger, APM (Datadog, New Relic, ELK), automação de incidentes (PagerDuty, OpsGenie).

### Fase 5 — MultiCloud e Segurança
Multicloud (AWS, Azure, GCP), SRE (SLA/SLI/SLO, MTTR, RTO/RPO), FinOps (rightsizing, savings plans, Kubecost), ITSM e AIOps.
